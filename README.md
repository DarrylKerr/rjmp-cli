# rimp-cli

Author: [RainJoy](https://github.com/zhouyu1993)

## Installation

``` bash
npm install -g rjmp-cli
# OR
yarn global add rjmp-cli

rjmp -v
# OR
rjmp --version
```

## Creating a Project

``` bash
rjmp init <name>
```

## dependencies

fs + path + child_process

commander + inquirer + download-git-repo + handlebars

chalk + log-symbols + ora
